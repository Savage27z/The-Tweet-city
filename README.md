# TweetCity

Every Twitter/X account becomes a 3D pixel-art skyscraper in an explorable city.

Inspired by [thegitcity.com](https://thegitcity.com).

## Run

```bash
npm install
npm run dev
```

Visit http://localhost:3000

## Stack

Next.js 14 (App Router) · React Three Fiber · Three.js · Tailwind · Zustand · Silkscreen font
